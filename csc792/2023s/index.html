CSC792: Spatial Data Minig

Course Description: This course focuses on a particular advanced topic of current academic interest in the domain of spatial, temporal, and spatiotemporal data management. Significant themes include clustering, association analysis, and classification using data mining and machine learning approaches. Additionally, students will learn how to use common data mining tools and how to develop geospatial applications. Students will learn about interdisciplinary research on spatial data mining and current industrial strategies for handling spatiotemporal data. The essential principles, techniques, and applications of intelligent data processing and analysis are covered in this course. It will provide an in-depth grasp of the main ideas and common data mining approaches. Effective data mining encompasses multiple principles from various branches of computer science, including machine learning, databases, artificial intelligence, pattern recognition, optimization, algorithms, parallel processing, and data visualization. This course is primarily for beginning graduate students interested in data analysis and its applications.

Course Objectives

This course will cover the basic techniques in data analytics including the preparation and manipulation of data for analysis. Overview of data mining algorithms in:

  • Spatial Databases
  • Spatial Concepts
  • Spatial Data Models
  • Spatial Query Languages
  • Spatial Access Methods
  • Spatial Storage and Indexing
  • Data Mining and Machine Learning
  • Regression
  • Clustering
  • K-means
  • Association Analysis
  • Probabilistic Modeling
  • Logistic Regression
  • Neural Network Methods
  • Use Of Support Vectors
  • Spatiotemporal Event Forecasting
