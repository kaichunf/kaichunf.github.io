CSC592: Introduction to Machine Learning

Course Description: Machine learning is a collection of approaches that enable computers to learn from data and experience without the need for humans to manually specify the intended behavior. The importance of machine learning to artificial intelligence as an area of study and an industry has grown. This course provides an overview of some of the most widely employed machine learning methods. In this course, we will cover nearest neighbors, which is the standard nonparametric model. Then, parametric models such as linear regression, logistic regression, softmax regression, and neural networks are examined. The course then transitions to unsupervised learning, focusing mostly on probabilistic models, principal components analysis, clustering methods such as K-means, and autoencoders. Prerequisites: 1) Basic knowledge in linear algebra, probability and statistics. and 2) Undergraduate course on algorithms and any programming course.

Course Objectives

This course will cover the basic techniques and algorithms in machine learning. More specifically, the following topics will be covered in this course:

  • Nearest Neighbours
  • Linear Regression
  • Logistic Regression
  • Multiclass Classification
  • Basic Optimization
  • Support Vector Machines
  • Decision Trees
  • Neural Networks
  • Convolutional Neural Networks
  • Sequential Models
  • Clustering
  • Probabilistic Models
  • EM Algorithm
  • Autoencoders
